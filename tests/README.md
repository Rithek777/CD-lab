# Tests

This directory will contain unit and integration tests for the allocation profiler.

Planned test coverage:

- parsing small MLIR files generated by Flang
- detecting `fir.allocmem` and matching `fir.freemem`
- classifying HLFIR/FIR allocation causes
- checking source location reporting
- verifying optimizer suggestions

The current CMake test only verifies that the placeholder executable starts.
